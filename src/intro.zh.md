
# 用锈煮

这个*锈菜谱*是一个简单的示例集合,演示了使用Rust生态系统中的板条箱完成常见编程任务的良好实践.

[Read more about *Rust Cookbook*](about.html)包括如何阅读这本书的提示,如何使用这些例子,以及关于约定的注释.

## 贡献

这个项目旨在让新的Rust程序员更容易做出贡献,并且是参与Rust社区的简单方法.它需要并欢迎帮助.详情见[MD的贡献.].
